  Tentang

Halo semuanya, saya seorang Fullstack Web Developer, Graphic Designer, dan Video Editor. Saya mengawali karir sebagai Java Developer tahun 2018 hingga sekarang. Sejak lulus kuliah, saya mengerjakan projek pembuatan website menggunakan Java dan PHP. Selain membuat website, saya juga terlibat dalam pembuatan design grafis dan video editor.

Kemampuan design grafis menunjang untuk mengembangkan aplikasi yang dibuat agar tampilannya menjadi estetik. Misalnya pembuatan ikon logo, background website, cover slide yang ada di aplikasi. Tidak hanya berfungsi dengan baik, aplikasi yang dibangun menjadi cantik dan user friendly.

Awalnya saya hanya menyukai pembuatan Front End karena saya menyukai design grafis. Tapi semakin lama saya juga mulai menyelami dunia Back End dan lambat laun mulai candu dengannya.
